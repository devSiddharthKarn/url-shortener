# URL Shortener Backend

A TypeScript-based backend service for URL shortening.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run in development mode:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Start production server:
```bash
npm start
```

## API Endpoints

- `GET /` - API information
- `GET /health` - Health check endpoint
